# algolib
競プロ用ライブラリ

## 注意
- C++17以上前提
  - AtCoderの言語アップデートが来たらC++20前提になるかも
- `using namespace std;` すると動かなくなる可能性あり
  - 多分大丈夫だと思う...
  - `namespace algolib` とかした方がいいのかな
